# Screenshots for Katabolt README

This folder contains screenshots used in the project's README.md.

## Required Screenshots:

1. **homepage.png** - Screenshot of the main sweet catalog page showing the product grid, search bar, and navigation.

2. **admin-dashboard.png** - Screenshot of the admin dashboard showing inventory management controls, add/edit/delete buttons.

3. **login.png** - Screenshot of the login page with email/password fields and authentication UI.

4. **user-dashboard.png** - Screenshot of the user dashboard showing profile information and order history.

## How to Add Screenshots:

1. Run your frontend application locally
2. Navigate to each page
3. Take screenshots (Cmd+Shift+4 on Mac, or use browser dev tools)
4. Save them in this folder with the exact filenames above
5. Commit and push to GitHub

## Image Specifications:
- Format: PNG or JPG
- Recommended size: 1200x800 pixels or similar
- Include browser chrome for context
- Ensure text is readable
